Prework Study Guide

HTML

  • Hypertext Markup Language (html)
  • Gives the content a structure.
  • One of three cornerstone languages used in web development
  • HTML is made up of elements which work together to provide structure for pages on the web
  • The following elements are text elements, image elements, header elements and list elements
  • some elements can be used to divide content on a page including: meta-related elements, content division elements and semantic elements
  • HTML is the first thing developers should think about when developing a webpage, you need a structure before you can think about the pretty stuff.
  • Most basic building block of website

CSS

  • Cascading Style Sheets (CSS)
  • One of three conrerstone languages used in web development
  • Used to describe a presentation of a website thats created with HTML
  • It is how each element is rendered on a webpage

git

  • Git is a developers tool used for source code management
  • Open source control system used to handle small or large projects efficiently
  • Used to track changes in the source code, which allows devs to work together
  • Allows dev to have the full history of their code repository locally
  • Git commands; Git status: checks what branch we are currently on, git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • lightweight language
  • One of three cornerstone languages used in web development
  • Used to provide logic and interactivity of a website
  • Add interactive behavior such as; zooming in and out or playing audio/video
  • it makes applications more dynamic and exciting
  • Javacript can also slow down the performance of the site
