HTML

  • Hypertext Markup Language (html)
  • Gives the content a structure.
  • One of three cornerstone languages used in web development
  • HTML is made up of elements which work together to provide structure for pages on the web
  • The following elements are text elements, image elements, header elements and list elements
  • some elements can be used to divide content on a page including: meta-related elements, content division elements and semantic elements
  • HTML is the first thing developers should think about when developing a webpage, you need a structure before you can think about the pretty stuff.
